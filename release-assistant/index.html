LaunchDarkly Release Assistant (pipelines)

Each user gets one card per environment representing a connection to the launchdarkly environment environment and sends the context to the launchdarkly environment.

Flag: feature-lead-widget
Requests sent: 0 Beta-only requests: 0
Run npm start in this folder and open http://localhost:3000/ (demo is at the site root). Edit context.json or env.json and save — the UI refreshes on poll (every few seconds). Use a local static server so fetch works (not file://). OpenTelemetry metrics/traces go to LaunchDarkly via the observability plugin (see observability in env.json); requires LaunchDarkly Observability on your account.
